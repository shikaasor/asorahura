## Codebase Patterns
- Use `npx tsc --noEmit` for typecheck validation (not `npm run build`)
- CSS modules pattern: Import as `styles` and apply via `className={styles.name}`
- Client components: Add `"use client"` directive when using framer-motion, hooks, or browser APIs
- Animation pattern: framer-motion with `motion.div` + `initial/animate/transition` props
- Typography: Playfair Display (serif) for headlines/h tags, Inter (sans-serif) for body/UI
- Design tokens: CSS custom properties defined in globals.css (--foreground, --background, --muted, etc.)
- Page sections: Wrapper with className + nested .container div for consistent max-width
- Grid layouts: 3 columns desktop (repeat(3, 1fr)), collapse to 1 column at 1024px breakpoint
- Section spacing: 10rem vertical padding, alternating backgrounds (white, #f9fafb)
- Fixed navigation requires pages to have top padding (currently 15rem on hero sections)
- Active link detection: Use `usePathname()` hook and conditional className comparison
